SCREENSHOT PLACEMENT GUIDE
==========================

Add your project screenshots to this folder with the following naming convention:

1. AI Documentation Tool:
   - ai-docs-1.png (main UI screenshot)

2. Election System:
   - election-1.png (dashboard view)
   - election-2.png (voting interface)

3. Data Mining & AI:
   - ai-ml-1.png (model training results)

Recommended image sizes:
- Aspect ratio: 16:10 or 16:9
- Minimum width: 800px
- Format: PNG or JPG

The portfolio will automatically display these images when you click
on the screenshot placeholders in the projects section.
